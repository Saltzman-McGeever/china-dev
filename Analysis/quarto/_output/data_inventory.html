## Overview

This analysis visualizes Chinese development projects around the world using data from AidData's China's Global Loans and Grants Dataset, Version 1.0.

### Total Projects
```{python}
from plots import create_flat_map_visualization

# Count unique projects by country
fig = create_flat_map_visualization(
    clg_dev,
    agg_col='AidData_Record_ID',
    agg_func='nunique',
    value_label='Number of Projects',
    color_scale='YlOrRd'
```
